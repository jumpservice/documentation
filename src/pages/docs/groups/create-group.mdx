# Create group

## About group

In JumpServer, a group is a collection of users who share common characteristics or roles. Groups help administrators efficiently manage permissions, access control, and resource allocation by organizing users into logical units.

## Create group

<Steps>
<div>
<Nav name="console" />
</div>

<div>
In the left menu, under the USERS section, click **<Icon name="UserGroup" /> Groups**.
</div>

<div>
Click **+ Create**.
</div>

<div>
In the **Name** field, type group name.
</div>

<div>
In the **Users** field, select users to be added to the group.
</div>

<div>
In the **Description** field, type a description for the group.
</div>

<div>
Click **Submit**.
</div>
</Steps>
