# Message template

Administrators can modify the corresponding notification templates as needed. The following types of message templates are currently supported for customization:
- Session sharing
- Different city login reminder
- OAuth binding reminder
- Create account successfully
- Reset password
- Reset password success
- Reset public key success
- Password is about expire
- Account is about expire
- Reset SSH Key
- Reset MFA
- User login reminder
- User login alert for asset

## Edit a message templates

<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="Remind" /> Notifications**.
</div>

<div>
On the right page, click **Msg template** tab.
</div>

<div>
In the **Name** field, select the message template to be edited.

Below the field, click "<Icon name="LucideMessageCircleQuestion" />Help" to view the available variables for the selected template.
</div>

<div>
Click **Submit**.
</div>

</Steps>
