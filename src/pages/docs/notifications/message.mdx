# Message

Administrators can configure and enable SMS service. Currently, the following SMS providers are supported:
- Alibaba cloud
- Tencent cloud
- Huawei cloud
- CMPP v2.0
- Custom type
- Custom (File)


## Enable SMS service

<Steps>
<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="Remind" /> Notifications**.
</div>

<div>
On the right page, click **Message** tab.
</div>

<div>
In the **SMS** field, check the box to enable SMS service.
</div>

<div>
In the **Provider** field, select the SMS provider.
</div>

<div>
In the **Code length** field, set the length of the verification code. The default is 4.
</div>

<div>
In the **Provider** section, configure the parameters required by the selected SMS provider.
</div>

<div>
Click **Submit**.
</div>

</Steps>
