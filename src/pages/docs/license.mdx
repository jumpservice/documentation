# Import license <Icon name="xpack-lg" />

<EnterpriseFeatureCta
  feature="Enterprise license management"
  description="Activate JumpServer Enterprise Edition to unlock advanced PAM capabilities, X-Pack enhancements, and commercial support."
/>

This topic describes how to import the license for the enterprise edition of JumpServer.

<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
Navigate to the **System settings > License**.
</div>

<div>
In the "Quick update" section on the right side of the page, locate the "Import license" option and click the "Import" button.
<img src="/images/docs/admin-guides/system-settings/license/import-license.png" alt="image" width="450"/>
</div>

<div>
In the "Import license" dialog box, click the "Choose file" button to select the license file.
</div>

<div>
Click **Confirm**.
</div>

</Steps>
