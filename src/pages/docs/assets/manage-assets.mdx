# Manage assets

This topic describes how to manage assets.

## View asset details
1. In JumpServer, navigate to **Console > Assets**.
2. In the Assets list, click the asset name to navigate to the details page.
3. On the "Basic" page (which is selected by default), where you can view both **basic information** and **special information** (if the asset has any).
