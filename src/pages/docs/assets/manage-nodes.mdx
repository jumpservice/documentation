# Manage nodes

This topic describes how to manage nodes and the relationship between assets and nodes.

## Create a node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Create node** option from the pop-up menu.
(*Note: Right-clicking on a blank area will not work, it must be performed on the node.*)
<img src="/images/docs/admin-guides/assets/node-create.png" alt="node-create" width="350"/>

3. Modify the name and press **Enter** to complete the creation.

## Rename a node

1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Rename node** option from the pop-up menu.
3. Modify the name and press **Enter** to complete the creation.

## Delete a node

1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Delete node** option from the pop-up menu.

## Search for a node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Hover over** the asset tree area, and a `search` icon will appear on the right side of the root node.
3. **Click** the `search` icon and **type** the keyword to search.
<img src="/images/docs/admin-guides/assets/node-search.png" alt="node-search" width="350"/>
4. The asset tree will display the search results with `Search` as the root node.

## Move node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Press and hold** the left mouse button to drag the node, then release the button to drop it under the target node. 
3. Click **OK** in the popup to complete the move.
<img src="/images/docs/admin-guides/assets/node-move.png" alt="node-move" width="350"/>

## Add assets to a node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Add assets to node** option from the pop-up menu.
3. Select the asset you want to add and click **Confirm** to complete. *(Note: The asset’s other nodes will still be retained.)*
<img src="/images/docs/admin-guides/assets/node-select-assets.png" alt="image" width="650"/>

## Move assets to a node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Move assets to node** option from the pop-up menu.
3. Select the asset you want to move and click **Confirm** to complete. *(Note: The asset’s other nodes will be removed.)*

## Remove assets from a node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Remove assets from node** option from the pop-up menu.
3. Select the asset you want to remove and click **Confirm** to complete.

## Refresh the hardware information of assets in the node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Update node assets hardware information** option from the pop-up menu.
3. A new tab will open automatically to show the task execution logs.

## Test the connection of assets in the node
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Test assets connectivity of node** option from the pop-up menu.
3. A new tab will open automatically to show the task execution logs.

## Only show the current node’s assets
1. In JumpServer, navigate to **Console > Assets > All**.
2. **Right-click** on the node and select the **Only show current node assets** option from the pop-up menu.
3. The asset list on the right will refresh and display only the assets under the current node (by default, clicking the node will display all assets under the current node and its subnodes).

## Type tree
1. In JumpServer, navigate to **Console > Assets > All**.
2. Clicking on **Type tree** will display the asset tree by **Category > Type > Platform**.
