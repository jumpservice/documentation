# Application

This topic describes how to create application integrations and retrieve account credentials stored in JumpServer via API.

## Create an application

<Steps>

<div>
<Nav name="pam" />
</div>

<div>
In the left menu, click **<Icon name="application" /> Applications**.
</div>

<div>
On the right page, click the **Applications** tab.
</div>

<div>
Above the table, click **Create**.
</div>

<div>
Type a **Name** for the application.
</div>

<div>
Optionally, upload **Logo** for the application.
</div>

<div>
Set account policy for the application. You can choose to use **All accounts**, **Specific accounts** and **Filter by attribute**.
</div>

<div>
Type the IP address or range allowed to access the application.
</div>

<div>
Optionally, type a **Description** for the task.
</div>

<div>
Click **Submit**.
</div>

</Steps>
