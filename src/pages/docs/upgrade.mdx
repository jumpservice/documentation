# Upgrade

This guide shows you how to upgrade JumpServer on your platform.

We recommend that you upgrade JumpServer often to stay current with the latest fixes and enhancements.

## One-line installation script

1. Change to the `/opt` directory and download the `v4.6.0` installation package.

```sh
cd /opt && wget https://github.com/jumpserver/installer/releases/download/v4.6.0/jumpserver-installer-v4.6.0.tar.gz
```

2. Extract the installation package and change into the directory.

```sh
tar -xf jumpserver-installer-v4.6.0.tar.gz && cd jumpserver-installer-v4.6.0
```

3. Perform the upgrade. After running the upgrade command, databases and configuration files will be automatically backed up during the process.

```sh
./jmsctl.sh upgrade
```

4. After the upgrade is completed, the service will not start automatically. You need to execute the following command to start the service.

```sh
./jmsctl.sh start
```
