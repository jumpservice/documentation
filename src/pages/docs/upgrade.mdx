# Upgrade guide for JumpServer

This topic describes how to upgrade JumpServer.

We recommend upgrading JumpServer regularly to access the latest designs and features.

## Upgrade guide for one-line installation

<Alert>
If you have installed JumpServer using the [Install with a one-line command](/docs/installation#install-with-a-one-line-command) guide, you can upgrade JumpServer by running the following command.
</Alert>

<Steps>


<div>
Log in to the JumpServer deployment server using the "root" or another user with superuser privileges.
</div>

<div>
Change to the `/opt` directory.
</div>

<div>
Download the latest installation package.
```sh
wget https://github.com/jumpserver/installer/releases/download/v4.7.0/jumpserver-installer-v4.7.0.tar.gz
```
</div>

<div>
Extract the installation package.
```sh
tar -xf jumpserver-installer-v4.7.0.tar.gz
```
</div>

<div>
Change to the extracted directory.
```sh
cd jumpserver-installer-v4.7.0
```
</div>

<div>
Perform the upgrade. You may need to wait a moment ☕️ ...
```sh
./jmsctl.sh upgrade
```
</div>

<div>
Start the JumpServer service.
```sh
./jmsctl.sh start
```
</div>

</Steps>
