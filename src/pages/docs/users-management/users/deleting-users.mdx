# Deleting users

## Deleting a user

On JumpServer, navigate to user page, click the `···` icon at the end of the row for the user you want to delete, then select `Delete`.
![Delete User Button](/images/docs/users/delete-user-button.png)

## Bulk deleting users

<Callout emoji="💡" type="info">
    Deleting all users is not allowed.
</Callout>

1. On JumpServer, navigate to user page.

2. Select the users you want to delete by checking the checkbox on the left side of the user’s row.
![Bulk Delete User](/images/docs/users/bulk-delete-user.png)

3. Click the `Actions` dropdown button in the top left and select `Delete selected`.
