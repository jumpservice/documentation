# Viewing users

## Viewing all users

1. On JumpServer, navigate to `Console > Users` page.

2. You can customize the columns displayed in the current table by using the settings button in the top right corner.
![View Users Settings](/images/docs/users/view-users-table-settings.png)

3. In the popup window, you can click `All` in the top left corner to display all fields,
or click 'Reset to default' in the bottom right corner to restore the default column display.
`Name`, `Username`, and `Actions` columns are mandatory and cannot be unchecked.
![View Users Settings Attributes](/images/docs/users/view-users-table-attributes.png)

4. By default, the table displays 15 entries per page.
You can adjust the number of entries per page using the settings in the bottom right corner of the table,
with options for 15, 30, 50, or 100 entries.
![View Users Settings Page](/images/docs/users/view-users-table-page.png)

## Searching users
In addition to viewing all users, you can also use fuzzy search, exact search, and tag filters to find users.

1. `fuzzy search` Type characters in the search box or press `/` to quickly focus.
The table will display users whose `Name`, `Username`, or `Email` contains the search characters.
![Search Users Fuzzy](/images/docs/users/search-users-fuzzy.png)

2. `exact search` Click the `⌄` to the left of the search box, select a user attribute field, and type the complete value to search.
Only full attribute values will yield successful results.
![Search Users Exact](/images/docs/users/search-users-attributes.png)

3. `Tag search` You can also filter users by tags.
![Search Users Tag](/images/docs/users/search-users-tag.png)
