# Editing users

## Editing a user

1. On JumpServer, navigate to user page.

2. Click the `Edit` button at the end of the user‚Äôs row.
![Edit User Button](/images/docs/users/edit-user-button.png)

3. Edit the user‚Äôs information.

4. Click the `Submit` button.

## Quickly editing users in the table

Currently, quick editing of users in the table only supports the Tags field.

1. On JumpServer, navigate to user page.

2. In the user table, find the `Tags` column.
![Edit User Table Tags](/images/docs/users/edit-user-table-tags.png)
If it's not visible, you need to check the `Tags` field in the `Table Preferences` at the top right.
![Edit User Table Tags Settings](/images/docs/users/edit-user-table-tags-settings.png)

3. Hover over the `Tags` field of the user, and an edit button will appear, click it.
![Edit User Table Tags Edit Icon](/images/docs/users/edit-user-table-tags-edit-icon.png)

4. A popup for `Associated tags` will appear on the current page, where you can directly select a tag key and tag value.
You can also type a new key and value to create and associate them immediately.
![Edit User Table Tags Input](/images/docs/users/edit-user-table-tags-input.png)

5. Click `Confirm` to complete the association.

6. If you want to remove associated Tags, simply click the `x` button on the tag.
![Edit User Table Tags Remove](/images/docs/users/edit-user-table-tags-remove.png)


## Bulk editing users

1. On JumpServer, navigate to user page.

2. Select the users you want to edit by checking the checkbox on the left side of the user‚Äôs row.
![Bulk Edit User Checkbox](/images/docs/users/bulk-edit-user-checkbox.png)

3. Click the `Actions` dropdown button in the top left and select `Update selected`.
![Bulk Edit User Actions](/images/docs/users/bulk-edit-user-actions.png)

4. In the popup window, you can select the attribute to modify in the first row.
Only the `Groups`, `Expiration Date`, and `Description` attributes can be modified.
![Bulk Edit User Attributes](/images/docs/users/bulk-edit-user-attributes.png)

5. Update the user information in the form below.

6. Click the `Confirm` button.
