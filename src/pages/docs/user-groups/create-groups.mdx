# Create groups

This topic describes how to create groups.

1. Navigate to **Console > Groups** and click the **+ Create** button.
  
  <img src="/images/docs/admin-guides/groups/create-group.png" alt="group-create-button" width="325"/>

2. In the **Name** field, enter the groupâ€™s name.
3. Optionally, in the **Users** field, you can assign exsited users to this group.
4. Optionally, in the **Description** field, enter a description of this group.
5. Click **Submit** to create the group, or **Save & Continue** to create and add more.
