# Users

This section describes users, and how to create and manage them in JumpServer.

## About
You can assign specific permissions and roles to different users to control their behavior in JumpServer.
Users are individuals accessing resources through JumpServer. Users authenticate to the system, inherit permissions through roles or groups, and perform authorized actions on assets.
