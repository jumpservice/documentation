# Introduction

Note: JumpServer documentation is continually being updated. Please stay tuned for more details.


## About JumpServer

**JumpServer** is an open-source Privileged Access Management (PAM) platform that provides DevOps and IT teams with on-demand and secure access to SSH, RDP, Kubernetes, Remote Apps and Database endpoints through a web browser.
