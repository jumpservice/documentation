# Create roles

There are two ways to create a role, Manually create and Import create.

## Manually create
### Creating  System Role
1. In JumpServer, navigate to **Console** > **USER** > **Roles**, to enter the **System roles** management interface and click the **Create** button.
![Roles](/images/docs/roles/System-roles.png)

2. In the **Name** field, enter a role name that is unique in the system.

3. In the **Scope**, System.

4. In the **Permissions**, after successful creation or update, add to the details.

5. In the **Description**, can be used to detail role functions, responsibilities, applicable scenarios, etc., to assist role management and identification.

### Creating Organizational Role
1. In JumpServer, navigate to **Console** > **USER** > **Roles** > **Organizational roles**, to enter the **Organizational roles** management interface and click the **Create** button.
![Roles](/images/docs/roles/Organizational-roles.png)

2. In the **Name** field, enter a role name that is unique in the organisation.

3. In the **Scope**, Organization.

5. In the **Description**, can be used to detail role functions, responsibilities, applicable scenarios, etc., to assist role management and identification.

## Import create
### Import  System Role
1. In JumpServer, navigate to **Console** > **USER** > **Roles**, to enter the **System roles** management interface and click the **Import** button.
![Roles](/images/docs/roles/System-roles-import.png)

2. Download creation template. The template supports two file formats, **CSV** or **XLSX**, choose any one to download.
![Roles](/images/docs/roles/System-roles-import-template.png)

3. Open the downloaded template form and fill in the role information.
![Roles](/images/docs/roles/System-roles-template-file.png)

4. Import the form with the role information filled in.
