# Create an application

**Applications** are used to allow external systems to call and retrieve accounts and passwords stored in JumpServer.

<Steps>

<div>
<Nav name="pam" />
</div>

<div>
In the left menu, click **<Icon name="Service" /> Applications**.
</div>

<div>
On the right page, click the **Applications** tab.
</div>

<div>
Click **+ Create**.
</div>

<div>
In the **Name** field, type the application name.
</div>

<div>
In the **Logo** field, upload the application logo.
</div>

<div>
In the **Account** field, select one of the options: "All accounts", "Specific accounts", or "Filter by attribute". 
</div>

<div>
In the **Access IP** field, type the IP address or range allowed to access the application.
</div>

<div>
In the **Active** field, check to enable the application.
</div>

<div>
In the **Description** field, type the application description.
</div>

<div>
Click **Submit**.
</div>

</Steps>
