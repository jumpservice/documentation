# Create account risk detection task <Icon name="xpack-lg" />

**Account risk detection** checks password length, repeated passwords, and common weak passwords. Supports custom execution cycles for regular scanning of all accounts in JumpServer.

<Steps>

<div>
<Nav name="pam" />
</div>

<div>
In the left menu, click **<Icon name="Scan" /> Risk detection**.
</div>

<div>
On the right page, click **Detection tasks** tab.
</div>

<div>
Click **+ Create**.
</div>

<div>
In the **Name** field, type the task name.
</div>

<div>
In the **Assets** field, select one or more assets. Detects accounts under the selected assets.
</div>

<div>
In the **Nodes** field, select one or more nodes. Detects accounts under assets in the selected nodes and their subnodes.
</div>

<div>
In the **Engines** field, select one or more check engines.
</div>

<div>
In the **Recipients** field, select one or more recipients. Sends the detection result after task completion.
</div>

<div>
In the **Periodic** field, check to enable scheduled execution.
</div>

<div>
In the **Active** field, check to enable the task.
</div>

<div>
In the **Description** field, type the task description.
</div>
    
<div>
Click **Submit**.
</div>
</Steps>
