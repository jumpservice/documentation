# Job center

Administrators can globally enable the Adhoc feature for all users and configure the command blacklist in Adhoc.

## Enable Adhoc
<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="Feature" /> Features**.
</div>

<div>
On the right page, click **Job center** tab.
</div>

<div>
In the **Adhoc command** field, check the box to enable Adhoc.
</div>

<div>
Click **Submit**.
</div>

</Steps>

## Configuration command blacklist

Administrators can configure the command blacklist in Adhoc to prevent users from executing risky commands on assets.

<Alert type="note">
This command blacklist takes effect only in **Workbench > JOB CENTER > Adhoc**.
</Alert>

<Steps>

<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="Feature" /> Features**.
</div>

<div>
On the right page, click **Job center** tab.
</div>

<div>
In the **Command blacklist** field, type the commands to be blacklisted, and press Enter after each command to confirm.
</div>

<div>
Click **Submit**.
</div>

</Steps>
