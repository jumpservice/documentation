# Virtual App <Icon name="xpack-lg" />

<EnterpriseFeatureCta
  feature="Virtual App publishing"
  description="Use Linux platforms as application delivery hosts to publish and connect to containerized apps through JumpServer."
/>

Administrators can enable the virtual application feature to use the Linux platform as an application service publishing platform.

## Enable virtual app

<Steps>
<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="Feature" /> Features**.
</div>

<div>
On the right page, click **VirtualApp** tab.
</div>

<div>
In the **Virtual app** field, check the box to enable Virtual app.
</div>

<div>
Click **Submit**.
</div>
</Steps>

## Configuration virtual app

For instructions on how to configure and use virtual applications, refer to [Enable and configure virtual app](/docs/remoteapp/virtualapp#enable-and-configure-virtualapp).
