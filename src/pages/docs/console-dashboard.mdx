# Console Dashboard

Navigate to `Console > Dashboard`.

![Dashboard](/images/docs/console-dashboard.png)

## Real-time
* **Online sessions**: Number of online asset sessions.
* **Online users**: Number of users connecting to assets.
* **Failed sessions today**: Number of failed sessions connecting to assets today.

## User / Asset activity

This chart shows the asset access activity over a week. It includes two smooth curves: 
One representing the `number of users` accessing assets daily and the other showing the `total number of assets` accessed each day.

## User data

	- **Total users**: Number of users in the current organization.
	- **Weekly add**: Number of newly created users in the current organization this week.
	- **Users logged today**: Number of users logged in today.
	- **Proportion**: Percentage of users logged in today compared to total users.

## Asset data

	- **Total assets**: Number of assets in the current organization.
	- **Weekly add**: Number of newly created assets in the current organization.
	- **Active assets today**: Number of assets connected today.
	- **Proportion**: Percentage of assets connected today compared to total assets.
  
## Asset type proportion

The asset type proportion shows the distribution of different types of assets within the current organization. Hovering the mouse over each type allows you to view the actual quantity of that asset type.

## Login user ranking

Top 10 users ranked by asset connection frequency, with options to view statistics for today, the past seven days, or the past thirty days.

## Login asset ranking

Top 10 assets ranked by connection frequency, with options to view statistics for today, the past seven days, or the past thirty days.
