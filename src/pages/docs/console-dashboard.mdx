# Console Dashboard

Navigate to `Console > Dashboard`.

![Dashboard](/images/docs/console-dashboard.png)

**Real-time**
* Online sessions: The total number of asset sessions.
* Online users: The total number of users currently connecting to assets.
* Failed sessions today: The total number of connection failures for all users accessing assets today.

**User/Asset activity**

This chart shows the asset access activity over a week. It includes two smooth curves: 
One representing the `number of users` accessing assets daily and the other showing the `total number of assets` accessed each day.

**User data**

	- Total users: Total number of users in the current organization.
	- Weekly add: Number of new users added this week.
	- Users logged today: Number of users logged in today.
	- Proportion: Percentage of users logged in today.

**Asset data**

	- Total assets: Total number of assets in the current organization.
	- Weekly add: Number of new assets added this week.
	- Active assets today: Percentage of users who logged in today.
	- Proportion: Percentage of assets connected today.
  
**Asset type proportion**

The asset type proportion shows the distribution of different types of assets within the current organization. Hovering the mouse over each type allows you to view the actual quantity of that asset type.

**Login user ranking**

Displays the top 10 rankings of users by asset connection frequency, with options to view statistics for `today`, the past `seven` days, or the past `thirty` days.

**Login asset ranking**

Displays the top 10 rankings of assets by connection frequency, with options to view statistics for `today`, the past `seven` days, or the past `thirty` days.
