# Delete users

<Alert>
If you only want to temporarily prevent a user from logging in, please refer to [Enable or disable users](/docs/users/enable-disable-users).
</Alert>

<Alert type="warning">
Deleting a user will also remove all associated authorizations and cannot be undone. Please confirm before proceeding.
</Alert>

## Delete a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, find the user that you want to delete.
</div>

<div>
In the Actions column, click the **···** icon, then click **Delete**.
</div>

<div>
Confirm the information and click **OK**.
</div>

</Steps>


## Bulk delete users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, check the users you want to delete.
</div>

<div>
Above the table, select **Actions <Icon name="chevron-down" />** button, then click **<Icon name="Trash2" /> Delete selected**.
</div>

<div>
Confirm the information and click **OK**.
</div>

</Steps>
