# Delete users

## Delete a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and locate the relevant user in the Users list.
</div>

<div>
Select the **···** dropdown menu in the `Actions` column, then click **Delete**. Alternatively, in the top-right corner of the user details page, click **Delete**.
</div>

<div>
Confirm the information to be deleted In the pop-up window, and then click **OK**.
</div>

</Steps>


## Bulk delete users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and select relevant users in the Users list.
</div>

<div>
Click the **Action** dropdown menu above the Users list, then select **Delete selected**.
<img src="/images/docs/admin-guides/users/bulk_delete_users.png" alt="image" width="290"/>
</div>


<div>
Confirm the information to be deleted In the pop-up window, and then click **OK**.
</div>

</Steps>
