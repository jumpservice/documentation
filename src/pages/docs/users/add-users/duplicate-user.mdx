# Duplicate a user

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users** and locate the relevant user in the Users list.
</div>

<div>
Select the <Icon name="..." /> dropdown menu in the Actions column, then click **Duplicate**.
<img src="/images/docs/admin-guides/users/duplicate-user.png" alt="user-create-button" width="310"/>
</div>

<div>
Modify the user properties and information, and then click **Submit**.
</div>

</Steps>
