# Invite users to organization

<Steps>

<div>
Select a specific organization in the **Organization Selector** on the top-left of the page.
<img src="/images/docs/admin-guides/users/select-org.png" alt="user-create-button" width="300"/>
</div>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users**, then click **Invite** above the Users list.
<img src="/images/docs/admin-guides/users/invite-user.png" alt="user-create-button" width="300"/>
</div>

<div>
Select relevant users you want to invite and assign organizational roles to the users to set their permission level in this organization.
<img src="/images/docs/admin-guides/users/invite-user-page.png" alt="user-create-button" width="550"/>
</div> 

<div>
Click **Confirm** to invite the users.
</div>
</Steps>
