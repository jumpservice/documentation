# Reset user password

## Send password reset email

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to the **User** and locate relevant user in the Users list.
</div>

<div>
Click the user Name to open the details page.
</div>

<div>
On the `Basic` page (default), locate the `Quick update` card on the right and click **Send** button on the right side of **Reset password**.
<img src="/images/docs/admin-guides/users/reset-password.png" alt="reset-sshkey" width="550"/>
Then a reset password email will be sent to the user.
</div>

</Steps>

## Manually reset user password

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to the **Users** and locate the relevant user in the Users list.
</div>

<div>
Click the **Edit** in the `Actions` column. Alternatively, in the top-right corner of the user details page, click **Edit**.
</div>

<div>
Change the password in section Authentication, and then click **Submit**.
</div>

</Steps>
