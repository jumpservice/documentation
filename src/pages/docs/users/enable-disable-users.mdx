# Enable or disable users

For security, users who havenâ€™t logged in for a long time are disabled. For more information about the configuration guide, see [Auto disable threshold (day)](/docs/security/auth-security#auto-disable-threshold-day).

## Enable or disable a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, click the name of the user you want to enable or disable.
</div>

<div>
On the user details page, click **Basic** tab.
</div>

<div>
In the Quick update card, turn on/off the "Active" toggle to **enable** or **disable** the user.
</div>

</Steps>

## Bulk enable or disable users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, check the users you want to enable or disable.
</div>

<div>
Above the table, select **Actions <Icon name="chevron-down" />** button, then click **<Icon name="CircleCheck" /> Activate selected** or **<Icon name="Ban" /> Disable selected**.
</div>

</Steps>
