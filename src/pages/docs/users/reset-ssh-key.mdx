# Reset user SSH key

## Send reset SSH key email

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and locate relevant user in the Users list.
</div>

<div>
Click the user Name to open the details page.
</div>

<div>
On the `Basic` page (default), locate the `Quick update` card on the right and click **Send** button on the right side of **Reset ssh key**.
<img src="/images/docs/admin-guides/users/reset-sshkey.png" alt="reset-sshkey" width="550"/>
Then a reset SSH key email will be sent to the user.
</div>

</Steps>
