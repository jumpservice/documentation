# Edit user
The edit user feature is used to modify user information, manage permissions, and control account status, helping administrators easily maintain users and manage system access.

## Edit a user

<Steps>
<div>
<Nav name="console" />
</div>

<div>
In the left menu, under the USERS section, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, find the user that you want to edit.
</div>

<div>
In the Actions column, click **Edit**.
</div>

<div>
Modify user information. About user parameters, see [User parameters reference](/docs/users/user-parameter-reference).
</div>

<div>
Click **Submit**.
</div>
</Steps>

## Bulk edit users


<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, check the users you want to edit.
</div>

<div>
Above the table, select **Actions <Icon name="chevron-down" />** button, then click **<Icon name="BatchUpdate" /> Edit selected**.
</div>

</Steps>
