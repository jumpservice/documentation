# Remove users from organization

## Remove a user

<Alert type="note">
Removed users are not deleted from the system, but from the current organization.
They can still log in to the system and access other organizations they belong to.
You can also find the removed users in the Global organization.
</Alert>

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and locate the relevant user in the Users list.
</div>

<div>
Select the **···** dropdown menu in the `Actions` column, then click **Remove**.
</div>

<div>
Confirm the information to be removed In the pop-up window, and then click **OK**.
</div>

</Steps>


## Bulk remove users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and select relevant users in the Users list.
</div>

<div>
Click the **Action** dropdown menu above the Users list, then select **Remove selected**.
<img src="/images/docs/admin-guides/users/bulk_remove_users.png" alt="image" width="290"/>
</div>


<div>
Confirm the information to be removed In the pop-up window, and then click **OK**.
</div>

</Steps>
