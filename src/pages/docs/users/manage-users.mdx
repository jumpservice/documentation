# Manage users

This topic describes how to manage users.

## View user details 

1. In JumpServer, navigate to **Console > Users**.
2. In the Users list, locate the relevant user and click the user name to open the details page.
3. On the `Basic` page (default), where you can view `Basic information` and more.

<img src="/images/docs/admin-guides/users/user-detail.png" alt="user-detail" width="550"/>

## Edit user properties and information 

1. In JumpServer, navigate to **Console > Users**.
2. In the Users list, locate the relevant user and click the **Edit** in the `Actions` column. Alternatively, in the top-right corner of the user details page, click **Edit**.
3. Change the properties or information that you want, and then click **Submit**.

## Delete a user
1. In JumpServer, navigate to **Console > Users**.
2. In the Users list, locate the relevant user and select the **···** dropdown menu in the `Actions` column, then click **Delete**. Alternatively, in the top-right corner of the user details page, click **Delete**.
3. Confirm the information to be deleted in the popup, and then click **OK**.

## Remove a user from current organization
1. In JumpServer, navigate to **Console > Users**.
2. In the Users list, locate the relevant user and select the **···** dropdown menu in the Actions column, then click **Remove**.
3. Confirm the information to be removed in the popup, and then click **OK**.
4. You can find the user you removed in global organization.

## Duplicate a user
1. In JumpServer, navigate to **Console > Users**.
2. In the Users list, locate the relevant user and select the **···** dropdown menu in the Actions column, then click **Duplicate**.
3. Modify the user properties and information, and then click **Submit**.

## Invite users
1. In JumpServer, navigate to **Console > Users**, then click **Invite** above the Users list.

<img src="/images/docs/admin-guides/users/invite-user.png" alt="user-create-button" width="300"/>

2. Select the relevant users you want to invite and assign organizational roles to the users to set their permission level in this organization.
  
<img src="/images/docs/admin-guides/users/invite-user-page.png" alt="user-create-button" width="550"/>

3. Click **Confirm** to invite the users.

## Enable or disable a user

1. In JumpServer, navigate to **Console > Users**.

2. In the Users list, locate the relevant user and click the user name to open the details page.

3. On the `Basic` page (default), locate the `Quick update` card on the right, then click **Active switch** to enable or disable the user.

<img src="/images/docs/admin-guides/users/enable-disable.png" alt="enable-disable-user" width="550"/>

## Reset password

1. In JumpServer, navigate to **Console > Users**.

2. In the Users list, locate the relevant user and click the user name to open the details page.

3. On the `Basic` page (default), locate the `Quick update` card on the right and click **Send** button on the right side of **Reset password**. A password reset email will be sent to this user.

<img src="/images/docs/admin-guides/users/reset-password.png" alt="reset-password" width="550"/>

## Assign user groups

1. In JumpServer, navigate to **Console > Users**.

2. In the Users list, locate the relevant user and click the user name to open the details page.

3. On the `Basic` page (default), locate the `Groups` card on the right. 
4. Select one or more groups, and you can search by group name, then click **Add**.

<img src="/images/docs/admin-guides/users/assign-group.png" alt="assign-group" width="325"/>

5. Click **-** icon to **Remove** the group.
