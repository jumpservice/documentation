# Edit users

## Edit a user
<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **Users** and locate the relevant user in the Users list.
</div>

<div>
Click the **Edit** in the `Actions` column. Alternatively, in the top-right corner of the user details page, click **Edit**.
</div>

<div>
Change the properties or information that you want, and then click **Submit**.
</div>

</Steps>

## Bulk edit users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and select relevant users in the Users list.
</div>

<div>
Click the **Action** dropdown menu above the Users list, then select **Update selected**.
<img src="/images/docs/admin-guides/users/bulk-edit-users.png" alt="image" width="290"/>
</div>

<div>
Modify the users properties and information, and then click **Submit**.
<Alert type="note">
Bulk edit users action is only available for the following properties:
1. **Groups**
2. **Expiration date**
3. **Description**
</Alert>
</div>

</Steps>
