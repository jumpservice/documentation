# Create users

This topic describes how to create users.

1. Navigate to **Console > Users** and click the **+ Create** button.
  
  <img src="/images/docs/admin-guides/users/create-user.png" alt="user-create-button" width="600"/>

2. In the **Name** field, enter the user’s name.
3. In the **Username** field, enter a unique username for logging into JumpServer.
4. In the  **Email** field, enter the user’s email address in the **Email** field to receive system messages. Make sure the email address is not shared by other users.
5. Optionally, in the **group** field, select user's group. Grouping users simplifies permission management and batch assignments.
6. In the **Password settings** field, choose between setting a manual password or generating a password reset email. Ensure the email server is configured in System Settings first if using the latter option.
7. Optionally, in the **MFA** field, select **Disabled**, **Enabled**, or **Force Enabled**.
8. Optionally, in the **Source**, specify the user's source (e.g., Database for manual creation, LDAP for imports).
9. In the **System Roles** field, assign user's system role to to define permissions in JumpServer (e.g., System Administrator, System Auditor, User).
10. In the **Organizational Roles** field, assign user's Organizational role to to define permissions under current organization (e.g., Org. Administrator, Org. Auditor, Org. User).
11. In the **Active** field, enable or disable the user.
12. In the **Date expired** field, set the user's validation period.
13. Optionally, in **Other** section, enter the user’s mobile phone number and description.
14. Click **Submit** to create the user, or **Save & Continue** to create and add more.
