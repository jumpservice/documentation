# View user authorizations

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to **Console > Users**. Then, click the target user's username to view the user's information.
</div>

<div>
In the right-hand panel, you can view sections like `Basic` information, `Authorized assets`, `Authorization rules`, `Login ACLs`, `Asset sessions`, and `Activities` associated with this user.
<img src="/images/docs/admin-guides/users/view-user-authorizations.png" alt="image" width="650"/>

- **Basic**: View the user's basic information.
- ​**Authorized assets**: View the list of resources the user has access to.  
- ​**Authorization rules**: View all permission policies assigned to the user.  
- ​**Login ACLs**: View the access control lists governing the user's login permissions.  
- ​**Asset sessions**: View current and past connections to managed assets.  
- ​**Activities**: View a chronological record of user actions and events.  
</div>

</Steps>
