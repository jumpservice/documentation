# Disable users

## Disable a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to the **User** and locate the relevant user in the Users list.
</div>

<div>
Locate the relevant user and click the user name to open the details page.
</div>

<div>
On the `Basic` page (default), locate the `Quick update` card on the right, then click **Active switch** to disable the user.
<img src="/images/docs/admin-guides/users/disable-user.png" alt="reset-MFA" width="550"/>
</div>

</Steps>

## Bulk disable users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
Navigate to the **User** and select relevant users in the Users list.
</div>

<div>
Click the **Action** dropdown menu above the Users list, then select **Disable selected**.
<img src="/images/docs/admin-guides/users/bulk_disable_users.png" alt="image" width="290"/>
</div>

</Steps>
