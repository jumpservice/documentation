## Import Users through Templates
1. Navigate to the `Console`
![console](/images/docs/users/navigate-console.png)
2. Select `Users > Users`, then click upload icon above the user list.
![click upload icon](/images/docs/users/upload-users-icon.png)

3. Download `create` template used to upload users(.csv/.xlsx).
![download template](/images/docs/users/download-users-template.png)

4. Provide the template requirements for filling in the user information. Format requirements will be listed in the templates.

5. Back to Import & Create page. Upload templates and click import.
![upload template](/images/docs/users/import-template.png)

## Export Users' Information 
1. Navigate to the `Console`
![console](/images/docs/users/navigate-console.png)
2. Search or select the Users you want to export in users list, then click export icon above user list.
![export icon](/images/docs/users/export-users-icon.png)
3. Select export range and click `Confirm` to export.
![export template](/images/docs/users/export-template.png)
