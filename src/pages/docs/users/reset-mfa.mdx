# Reset user MFA

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to the **User** and locate relevant user in the Users list.
</div>

<div>
Click the user Name to open the details page.
</div>

<div>
On the `Basic` page (default), locate the `Quick update` card on the right and click **Send** button on the right side of **Reset MFA**.
<img src="/images/docs/admin-guides/users/reset-sshkey.png" alt="reset-MFA" width="550"/>
Then after user login, the user will be able to reconfigure MFA.
</div>

</Steps>
