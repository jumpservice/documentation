# Activate users

For security, users who havenâ€™t logged in for a long time are disabled. The administrator can activate them by following the steps below.

## Activate a user

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, click the name of the user you want to activate.
</div>

<div>
On the user details page, click **Basic** tab.
</div>

<div>
In the Quick update card, turn off the **Active** toggle to activate the user.
</div>

</Steps>

## Bulk activate users

<Steps>

<div>
<Nav name="console" />
</div>

<div>
In the left menu, click **<Icon name="UserO" /> Users**.
</div>

<div>
In the user table, check the users you want to activate.
</div>

<div>
Above the table, select **Actions <Icon name="chevron-down" />** button, then click **<Icon name="CircleCheck" /> Activate selected**.
</div>

</Steps>
