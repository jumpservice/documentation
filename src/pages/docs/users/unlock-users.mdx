# Unlock users

## Unlock a user

<Alert type="note">
To protect user accounts from unauthorized access, users are automatically locked after a certain number of failed login attempts.
To set the number of failed login attempts, go to **System Settings** > **Security** > **Login restriction** > **Login failures count**.
</Alert>

<Steps>
<div>
<Nav name="console" />
</div>

<div>
Navigate to **User** and locate relevant user in the Users list.
</div>

<div>
Click the user Name to open the details page.
</div>

<div>
3. On the `Basic` page (default), locate the `Quick update` card on the right and click **Unlock** button on the right side of **Unlock User**.
<img src="/images/docs/admin-guides/users/unlock-user.png" alt="unlock-user" width="550"/>
</div>

</Steps>
