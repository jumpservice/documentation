## Configuration parameters

### Site URL

`required`

This setting is used to generate the externally accessible JumpServer URL when required (for example, to create clickable links in notification messages).

### Document URL

Custom documentation URL. 

In the navigation bar, users can view it by clicking <Icon name="QuestionMark" /> > Docs.

### Support URL

Custom support URL.

In the navigation bar, users can view it by clicking <Icon name="QuestionMark" /> > Support.
