# Components

For more information about JumpServer components, see [About components](/docs#about-components).

## Set the component’s replay storage

<Steps>
<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="TerminalSet" /> Components**.
</div>

<div>
On the right page, click **Components** tab.
</div>

<div>
Find the component whose replay storage you want to set.
</div>

<div>
In the Actions column, click **Edit**.
</div>

<div>
In the **Replay storage** field, select the desired object storage from the dropdown list.
</div>

<div>
Click **Submit**.
</div>
</Steps>

## Delete offline components

<Steps>
<div>
<Nav name="system-settings" />
</div>

<div>
In the left menu, click **<Icon name="TerminalSet" /> Components**.
</div>

<div>
On the right page, click **Components** tab.
</div>

<div>
Find the offline component you want to delete.
</div>

<div>
In the Actions column, click **Delete**.
</div>

<div>
In the confirmation dialog, click **OK** to delete the component.
</div>
</Steps>
